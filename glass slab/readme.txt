GLASS SLAB EXPERIMENT
1. DESCRIPTION & USAGE: - This module demonstrates the refraction of light through a glass slab. -
 User can select and drag 'Pin_P1' on the board in real-time. - The incident laser ray emerges directly from Pin_P1. 
- As Pin_P1 moves, the incident angle changes 



. WHAT WORKS PERFECTLY: - Real-time dragging of Pin_P1 from a Top-Down (Ceiling Fan) Camera perspective. - Fully compatible with the New Input System framework. 
- Automated raycasting, refraction formulas, and geometric placement of reference pins.



3. WHAT COULD BE ADDED / FUTURE SCOPE: - An interactive UI Protractor to measure angles mathematically. - Dynamic text displays to show precise values for Angle of Incidence (i) and Angle of Refraction (r).
